# UNIX ENV
----

### Installation
Make sure zsh, vim, tmux already installed.   

- Quick install   
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/BobLiu20/unix_env/master/install_online.sh)"
```

- Install from source   
```
git clone https://github.com/BobLiu20/unix_env.git
cd unix_env
./install.sh
```
